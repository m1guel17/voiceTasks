About the Mock Provider

VoiceTasks ships with a built-in mock provider that works with zero API keys. It returns a sample transcription and pre-extracted tasks so you can explore the full UI immediately. When you add a real ASR or LLM provider above and mark it as active, the mock is automatically bypassed.

Supported ASR providers

  • Mock (default, no key needed)
  • OpenAI Whisper
  • Azure Speech Services
  • Google Cloud Speech-to-Text
  • Custom OpenAI-compatible endpoint

Supported LLM providers

  • Mock (default, no key needed)
  • OpenAI (GPT-4o, GPT-4, GPT-3.5)
  • Anthropic Claude
  • DeepSeek, Qwen, Groq, Mistral
  • Google Gemini, Azure OpenAI, Ollama
